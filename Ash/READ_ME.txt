Link the Ash rig in your blender scene

1. link the rig:
	File-> Link -> ash.blend -> Collection -> Ash-rig -> Link

2. Make a proxy
	With the ash rig selected in your scene, F3 -> type "Make Proxy", select this -> 
	In the new pop up window select rig

3. Run script
	Scripting workspace -> click folder icon -> select "rig_ui.py" -> hit play button(run script)

4. Have Fun!
	now select the rig controls and enter pose mode and have fun animating!
